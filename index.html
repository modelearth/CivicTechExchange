common/components/common/projects/ProfileProjectSearch.jsx

Install locally:

We may not need to do the following for our static setup:

python3 -m venv .env
source .env/bin/activate
pip install -r requirements.txt

If you are using Postgresql locally, you may need to remove these two lines in requirements.txt:

psycopg2==2.8.4
psycopg2-binary==2.8.4
